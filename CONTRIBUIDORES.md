# Contribuidores

Todo el que pasa por el onboarding se agrega aquí. Es tu primer PR: la prueba de que ya
sabes el flujo de branch → commit → push → pull request.

## Cómo agregarte

Añade **una línea nueva al final de la lista**, con el mismo formato que las de arriba:

```
- Tu Nombre — @tu-usuario-github — AAAA-MM-DD
```

Una rama por persona, un commit, un PR. No edites las líneas de los demás.
